## laf-cli

### Install

```bash
npm i laf-cli -g
```

## Usage 

#### 1. Generate a env file template for laf-cli: `.env`
```bash
  laf-cli init --access-key xxxxx --access-secret xxxxxx --bucket xxxxxx
```

#### 2. run `laf-cli sync ./dist` command to sync files to OSS bucket


## Upgrade laf-cli
```sh
  npm i laf-cli@latest -g
```
